# image_processing
基于OpenCv + Qt5 开发的图像处理小工具
## 开发环境
+ 系统环境：Manjaro Linux
+ 图形界面：Qt5
+ 图形处理：OpenCv 4.5.5
+ 构建工具：Xmake
+ 代码编辑：Vscode
+ 编程语言：C++11
